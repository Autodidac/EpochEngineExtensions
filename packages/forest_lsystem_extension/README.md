Optional Forest Factory extension source.

The core engine can eventually own the live Forest Factory workspace and runtime
activation path. This package stages optional L-system/reference material and a
deterministic segment generator that can feed future voxel occupancy, impostor,
mesh, or renderer-adapter outputs.
